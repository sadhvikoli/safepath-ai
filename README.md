# SafePath AI

SafePath AI is a privacy-first women's safety decision-support agent built with Gemini and FastAPI.

## Features

- AI-powered risk assessment
- Safety action planning
- Emergency resource recommendations
- Trusted-contact alert preparation
- Downloadable safety summaries
- Human-in-the-loop confirmation flow

## Stack

Frontend:
- React

Backend:
- FastAPI
- Gemini 2.5 Flash

## Run Backend

```bash
uvicorn main:app --reload
